== Description ==

Zillow Reviews Free Plugin by We Connect Code.

= Usage =

Display Review Anywehere You want in your website using sortcode.

== Installation ==

1. Upload the `plugin` folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Add shortcode [zillow_review] to anywhere you wish to display Expanded Ad Tool
